Les paramètres permettent de spécifier certaines options de l'application :
  • LIEN_FOOTER

    Lien situé dans le pied de page. Vous pouvez en positionner autant que souhaité.
    Le nom est utilisé comme texte pour le lien, le title est utilisé lors du survol de la souris, et l'url est utilisée pour le lien lui-même.

  • LOGO_FOOTER

    Lien vers une image qui sera affichée dans l'en-tête et le pied de page de l'application.
    Le nom est utilisé comme texte alternatif pour l'image et l'url est utilisée pour afficher l'image.

  • ENTITY_PICTURE

    Lien vers l'image affichée sur la page d'accueil.
    Le nom est utilisé comme texte alternatif pour l'image et l'url est utilisée pour afficher l'image.

  • SUPPORT_EMAIL

    Adresse mail du support.
    Le nom est utilisé comme texte alternatif pour l'adresse et l'url est utilisée pour l'adresse mail.

  • JOURS_ECHEANCE

    Nombre de jours avant l'échéance d'un questionnaire pour lequel un mail d'alerte est envoyé.
    Le nom est utilisé comme nombre de jours (entier). Si aucun paramètre n'est créé, alors le nombre de jours par défaut est 7.
